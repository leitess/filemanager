# Como testar?
Para os testes de integração realizado no cypress, é necessario que o frontend esteja rodando em paralelo:

```bash
cd client
npm run dev
npm run test
```

Para rodar os testes, usando a interface gráfica do Cypress use o comando abaixo:

```bash
cd client
npm run dev
npm run test:ui
```
